# usage_metrics_collect

Collect AI tool usage metrics for the specified period.

## Usage
Specify a time period and metric type to collect AI tool usage statistics.

## Parameters
- period: Time period to collect metrics for (e.g., "2026-01", "last_7_days")
- metric_type: Type of metric to collect (e.g., "tokens", "requests", "cost")
